Síntesis

La ontología STELLDJ está diseñada para modelar información clave del dominio automovilístico de las marcas de Stellantis, abordando aspectos como los segmentos de vehículos, tipos de motor, combustibles, gamas de precios y concesionarios. Esta ontología proporciona una estructura semántica que permite la interoperabilidad de datos y el análisis eficiente de información relevante para fabricantes, concesionarios y consumidores. El modelo incluye clases principales como Vehículo, Motor, Concesionario, EtiquetaMedioambiental y Segmento, conectadas mediante relaciones definidas y propiedades de datos que reflejan el mercado automovilístico real. Además, la ontología ha sido enriquecida con información externa obtenida de recursos como Km77 (especificaciones técnicas de vehículos), la DGT (etiquetas medioambientales) y Wikipedia (descripciones mediante rdfs:comments).

Introducción volver a índice

La ontología STELLDJ modela datos automovilísticos de las marcas de Stellantis, como segmentos de vehículos, tipos de motor, combustibles y concesionarios. Su objetivo es facilitar la interoperabilidad y el análisis eficiente del mercado automovilístico. Desarrollada con Chowlk, Protégé y documentada con Widoco, ofrece una estructura clara y reutilizable para aplicaciones semánticas.
